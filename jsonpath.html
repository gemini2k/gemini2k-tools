JSON 경로 추출기

JSON에서 점 표기 경로로 원하는 값만 뽑아냅니다. API 응답을 탐색하거나 디버깅할 때 쓰세요.

📖
사용 방법
  1. JSON을 붙여 넣습니다
  2. 경로를 입력합니다. 예: items[0].name, users.*.email
  3. 일치하는 값이 아래에 표시됩니다(여러 개면 배열로)

예시: users[0].name · users[*].email · users.*.name

점(.)으로 키를, 대괄호([0])로 배열 인덱스를 지정합니다. * 또는 [*]는 배열·객체의 모든 항목을 펼칩니다. 결과가 여러 개면 배열로 모아 보여 줍니다. 모든 처리는 브라우저에서만 이루어집니다.
