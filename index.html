설치 없이, 브라우저에서 안전하게 쓰는 무료 온라인 도구

빛의여행(gemini2k) 도구는 회원가입이나 프로그램 설치 없이 웹브라우저에서 바로 사용하는 무료 온라인 도구 모음입니다. 글자수 세기, 단위·좌표 변환, QR·비밀번호 생성, 이미지·PDF 처리, JSON·정규식 같은 개발 도구, 그리고 좌표계 변환·공간 연산 같은 GIS 전문 도구까지 한곳에서 제공합니다.

모든 계산과 변환은 이용자의 브라우저 안에서만 이루어지며, 입력한 텍스트나 업로드한 파일은 서버로 전송되거나 저장되지 않습니다. 자소서·리포트의 글자수를 확인하거나 민감한 문서를 변환할 때도 데이터가 외부로 나가지 않아 안심하고 사용할 수 있습니다.

주요 특징

  • 설치 불필요 — 브라우저만 있으면 PC·모바일 어디서나 즉시 사용
  • 개인정보 보호 — 입력 데이터는 기기에서만 처리, 서버 전송 없음
  • 완전 무료 · 다국어 — 한국어 / English / 中文 / 日本語 지원

제공하는 도구 분야

  • 텍스트 — 글자수 세기, 텍스트 정리(대소문자·중복 제거·정렬)
  • 계산기·변환 — 만나이·디데이, 단위·색상·음양력 변환
  • 생성기 — QR 코드, 안전한 비밀번호, 로또 번호
  • 이미지·PDF — 이미지 압축·리사이즈, PDF 합치기·쪼개기·압축, 이미지→PDF
  • 개발자 — JSON 포매터·검증, 정규식 테스트, 마크다운 편집, Base64
  • GIS·지도 — 좌표계 변환, 웹지도 거리·면적 측정, WKT/GeoJSON 변환, 공간 연산, 위치 비식별
  • 개인정보 — 개인정보 탐지·마스킹, 재식별 위험 평가

자주 묻는 질문

Q. 입력한 내용이 서버에 저장되나요?
아니요. 모든 처리는 브라우저 안에서 이루어지며 입력 데이터는 전송·저장되지 않습니다.

Q. 회원가입이나 로그인이 필요한가요?
필요 없습니다. 페이지에 접속하면 바로 사용할 수 있습니다.

Q. 비용이 드나요?
모든 도구는 무료입니다. 사이트 운영을 위해 광고가 표시될 수 있습니다.

쿠키 및 광고에 관한 자세한 내용은 개인정보처리방침을 참고하세요.
