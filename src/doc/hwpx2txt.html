한글(HWPX) 텍스트 추출기

한글 신형 형식(.hwpx)에서 본문 텍스트를 추출합니다. 한컴오피스 없이 내용을 바로 확인하세요.

📖
사용 방법
  1. 한글 파일을 .hwpx로 저장합니다 (한글에서 ‘다른 이름으로 저장 → HWPX’)
  2. 파일을 끌어다 놓거나 클릭해 올립니다
  3. 본문 텍스트가 추출되면 ‘복사’ 또는 ‘.txt 다운로드’
⚠️ 구형 .hwp는 한컴 독점 이진 형식이라 브라우저에서 안정적으로 읽을 수 없습니다. 한글에서 다른 이름으로 저장 → 한글 표준 문서(.hwpx)로 저장한 뒤 올려 주세요. (.hwpx는 표준 XML 압축 형식입니다)
📄
한글 파일(.hwpx)을 끌어다 놓거나 클릭
.hwpx (구형 .hwp는 .hwpx로 저장 후)
.hwpx 내부의 본문(섹션) 텍스트를 순서대로 추출합니다. 표·머리말 등 일부 요소는 단순 텍스트로 합쳐질 수 있고 서식·이미지는 제외됩니다. 파일은 서버로 전송되지 않고 브라우저 안에서만 처리됩니다.
