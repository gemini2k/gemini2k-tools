PDF → 이미지(PNG) 변환기

PDF의 각 페이지를 PNG 이미지로 바꿉니다. 발표 자료·문서를 이미지로 공유하거나 캡처할 때 좋아요.

📖
사용 방법
  1. PDF 파일을 끌어다 놓거나 클릭해 올립니다
  2. 화질(해상도)을 고르면 페이지별 이미지가 생성됩니다
  3. 각 페이지를 '저장'하거나 '전체 저장'으로 한꺼번에 내려받습니다
🖼️
PDF 파일을 끌어다 놓거나 클릭
.pdf
화질을 높이면 더 선명하지만 시간과 메모리를 더 씁니다. 페이지가 매우 많은 PDF는 처음 50페이지까지 변환합니다. 파일은 서버로 전송되지 않고 브라우저 안에서만 처리됩니다.
