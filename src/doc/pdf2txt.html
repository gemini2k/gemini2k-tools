PDF 텍스트 추출기

PDF에서 텍스트를 페이지 순서대로 추출합니다. 보고서·논문·계약서 내용을 빠르게 가져오세요.

📖
사용 방법
  1. PDF 파일을 끌어다 놓거나 클릭해 올립니다
  2. 페이지별 텍스트가 추출됩니다 (큰 파일은 잠시 걸릴 수 있어요)
  3. '복사' 또는 '.txt 다운로드'로 가져갑니다
📕
PDF 파일을 끌어다 놓거나 클릭
.pdf
텍스트가 들어 있는 PDF의 글자만 추출합니다. 스캔(이미지)으로 만든 PDF는 글자 정보가 없어 추출되지 않습니다(그런 경우 문자인식 OCR이 필요). 파일은 서버로 전송되지 않고 브라우저 안에서만 처리됩니다.
