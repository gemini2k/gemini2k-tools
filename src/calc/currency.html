환율 계산기

환율을 직접 입력해 금액을 환산합니다. 인터넷 통신 없이 입력한 환율로만 계산하므로 빠르고 안전합니다.

📖
사용 방법
  1. 두 통화 코드(예: USD, KRW)와 1단위당 환율을 입력합니다
  2. 위·아래 금액 칸 아무 곳에나 숫자를 넣으면 반대쪽이 자동 환산됩니다
  3. 환율은 은행·포털에서 확인한 값을 직접 넣으세요(실시간 아님)
1 USD = KRW
실시간 환율이 아닙니다 — 직접 입력한 환율로만 계산합니다(외부 서버 호출 0). 정확한 환율은 거래 은행이나 포털에서 확인하세요. 모든 계산은 브라우저에서 이루어집니다.
