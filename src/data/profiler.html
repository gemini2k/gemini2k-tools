데이터 프로파일러 (EDA)

CSV를 붙여넣거나 파일을 올리면 열마다 자료형·결측치·고유값·통계와 상위 빈도값을 한눈에 보여줍니다. 고유값이 많은 열은 준식별자 후보로 표시되어 재식별 위험 점검의 출발점이 됩니다.

📖
사용 방법
  1. CSV를 붙여넣거나 파일을 끌어다 놓습니다
  2. 첫 행이 머리글이면 '첫 행을 헤더로'를 켭니다 (구분자는 자동 감지)
  3. '분석'을 누르면 열별 통계와 준식별자 후보가 표로 나옵니다
📄 CSV 파일을 끌어다 놓거나 클릭해서 선택 (또는 아래에 붙여넣기)
자료형은 비어 있지 않은 값으로 추정합니다(정수·숫자·날짜·불리언·문자열). 준식별자 후보는 고유값 비율이 높아(기본 90% 이상, 2건 초과) 개인을 특정할 수 있는 열입니다 — 재식별 위험성 평가기로 이어서 점검해 보세요. 모든 분석은 브라우저에서만 이루어지며 데이터는 전송되지 않습니다.
