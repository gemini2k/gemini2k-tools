CSV 정제기

지저분한 CSV를 깔끔하게. 공백·중복 행·빈 행을 정리하고 헤더 이름을 정규화하며 구분자까지 바꿉니다. 구분자는 자동으로 감지합니다.

📖
사용 방법
  1. CSV를 붙여넣거나 파일을 올립니다
  2. 적용할 정리 옵션(공백·중복·빈행·헤더)을 고릅니다
  3. '정제'를 누르고 결과를 복사하거나 CSV로 내려받습니다
📄 CSV 파일을 끌어다 놓거나 클릭 (또는 아래에 붙여넣기)
↓
중복 행은 모든 열 값이 같은 행을 처음 1건만 남기고 제거합니다. 헤더 정규화는 공백을 _로 바꾸고 소문자화하며 중복 이름에 번호를 붙입니다(name, name_2). 빈 값 채우기는 비어 있는 셀을 지정 값으로 대체합니다. 모든 처리는 브라우저에서만 이루어집니다.
