CSV 조인·결합기

두 CSV를 공통 키 열을 기준으로 합칩니다(엑셀 VLOOKUP과 같은 개념). 표 A에 표 B의 정보를 붙일 때 쓰세요.

📖
사용 방법
  1. 표 A와 표 B의 CSV를 각각 붙여 넣습니다(첫 행은 머리글)
  2. 두 표를 잇는 공통 키 열을 각각 고릅니다(예: 사번, ID)
  3. 조인 방식을 고르고 '결합' → 미리보기 확인 후 CSV로 내려받습니다
키 값이 같은 행을 짝지어 A 열 + B 열로 합칩니다. B에 같은 키가 여러 개면 첫 번째를 사용합니다. 같은 열 이름이 겹치면 B 쪽에 _b를 붙입니다. 모든 처리는 브라우저에서만 이루어집니다.
