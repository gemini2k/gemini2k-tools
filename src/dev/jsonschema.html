JSON 스키마 생성기

예시 JSON을 붙여넣으면 그 구조에 맞는 JSON Schema(draft-07)를 자동으로 만들어 줍니다. API 문서·데이터 검증에 쓰세요.

📖
사용 방법
  1. 예시 JSON(객체 또는 배열)을 붙여넣습니다
  2. 타입·중첩·배열 항목을 분석해 스키마가 자동 생성됩니다
  3. ‘복사’로 가져가 검증 라이브러리(ajv 등)에 사용하세요
↓
값으로부터 타입을 추론합니다(정수/숫자/문자열/불리언/객체/배열/null). 배열은 첫 요소로 항목 스키마를 만들고, 객체는 키별로 분석합니다. 추론 결과이므로 필요에 맞게 다듬어 쓰세요. 모든 처리는 브라우저에서만 이루어집니다.
