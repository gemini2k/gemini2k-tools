CyberChef

데이터 인코딩·암복호화·포맷 변환의 “만능칼”. 레시피를 쌓아 Base64·해시·암호화·JWT·정규식 등 수백 가지 작업을 조합하세요.

📖
사용 방법
  1. 왼쪽 Operations에서 원하는 작업을 검색해 가운데 Recipe로 끌어다 놓습니다
  2. 아래 Input에 데이터를 넣으면 Output에 결과가 실시간으로 나옵니다
  3. 여러 작업을 순서대로 쌓아 복합 변환(예: URL디코드 → Base64디코드 → JSON정렬)을 만들 수 있어요
