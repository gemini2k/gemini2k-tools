SQL 포매터

한 줄로 뭉친 SQL을 절(SELECT·FROM·WHERE·JOIN…) 단위로 줄바꿈하고 들여쓰기해 읽기 좋게 정리합니다.

📖
사용 방법
  1. SQL 쿼리를 붙여넣습니다
  2. 키워드 대소문자 옵션을 고르면 자동으로 정렬됩니다
  3. ‘복사’로 정리된 쿼리를 가져갑니다
↓
절 단위(SELECT·FROM·WHERE·GROUP BY·ORDER BY·HAVING·LIMIT·JOIN·UNION 등)로 줄을 나누고 AND/OR는 들여씁니다. 작은따옴표 문자열은 보호되어 변형되지 않습니다. 복잡한 서브쿼리는 가독성 위주의 근사 정렬입니다. 모든 처리는 브라우저에서만 이루어집니다.
